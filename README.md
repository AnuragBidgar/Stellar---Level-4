# Decentralized Freelance Escrow and Reputation System

A production-ready decentralized application (dApp) built on Stellar Soroban smart contracts. This platform provides a Decentralized Freelance Escrow and Reputation System where clients can securely lock payments and freelancers are guaranteed payment upon successful job completion, driven by a trustless smart contract infrastructure.

## Architecture
- **Escrow Contract**: Handles job creation, payment locking, and fund release.
- **Reputation Contract**: Manages user trust scores based on completed jobs.
- **TRUST Token**: Custom Soroban token used for payments and staking.
- **Frontend**: React and Tailwind CSS application with Freighter wallet integration.
